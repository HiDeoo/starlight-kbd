## Import

```tsx
import { Kbd } from 'starlight-kbd/components'
```

## Usage

Use the `<Kbd>` component to display keyboard shortcuts in your documentation.

Different shortcuts can be provided for all the supported [keyboard types](/configuration/#types-configuration) configured in your documentation.
At least one shortcut matching the [`default`](/configuration/#default) keyboard type should be provided.

When a keyboard type is not provided, the default keyboard type is used.

<Preview>

```mdx
import { Kbd } from 'starlight-kbd/components'

Use <Kbd mac="Command+S" windows="Control+S" /> to get started.
```

Check the [shortcut syntax guide](/guides/shortcut-syntax/) for more information on how to write keyboard shortcuts.

## `<Kbd>` Props

The `<Kbd>` component accepts any number of props representing keyboard shortcuts for different [keyboard types](/configuration/#types-configuration) with at least one matching the [`default`](/configuration/#default) keyboard type.
